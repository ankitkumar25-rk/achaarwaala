## 4. OPERATIONAL DECISIONS

### 4.1 Logistics — Shiprocket (CONFIRMED)
- **Provider**: Shiprocket
- **Couriers available**: Delhivery, XpressBees, Ekart, BlueDart, Amazon ATS

### 4.2 Packaging Plan

| Item | Cost |
|------|------|
| PET jar (500ml, leak-proof, food-grade) | ₹12–18 |
| Induction seal liner | ₹2–3 |
| Custom label (FSSAI no. + MFG/EXP date) | ₹5–8 |
| Bubble wrap (2–3 layers) | ₹4–6 |
| Corrugated box | ₹8–12 |
| **Total** | **₹33–49 per unit** |

### 4.3 Compliance
- **FSSAI Basic Registration**: Apply at foscos.fssai.gov.in — ₹100/year
- **GST**: Optional below ₹40L annual revenue initially
- **Label requirements**: Product name, net weight, ingredients, MFG/EXP dates, FSSAI no., address

### 4.4 Inventory Strategy
- MOQ from factory: 50kg TOTAL (not per variety) — at ₹150/kg = ₹7,500
- Phase 1: Start with 10 bestselling varieties, 5kg each = 50kg
- Shelf life: 3–6 months → max 30-day stock at a time
- Storage: Home/family shop (zero cost)

### 4.5 Bhai's Network (EduMeasy Advantage)
- EduMeasy has school networks across India
- Corporate gifting: Teacher's Day, Diwali, Annual Day
- 1 school = 50 potential orders → 10 schools = 500 orders
- **Fastest revenue source in first 3 months** — faster than Instagram cold traffic

## 5. TECH STACK — WEBSITE

### 5.1 Source Project: PhotowalaGift (being cloned)
Ankit's existing production e-commerce project. Full stack reuse.

### 5.2 Tech Stack

**Backend:**
- Node.js (>=22) + Express.js (v5)
- **Prisma ORM + PostgreSQL** *(NOT MongoDB — important)*
- PASETO tokens + httpOnly cookie rotation (15min access / 7day refresh)
- Double-submit CSRF tokens
- Passport.js (Google OAuth 2.0)
- Valkey/Redis via ioredis (cache + rate limiting)
- Zod (validation)
- Resend SDK (email)
- Cloudinary (images)
- Razorpay (payments + webhooks)
- Google Maps Places API + Google Address Validation API
- Shiprocket API (logistics)

**Frontend:**
- React 19 + Vite
- Tailwind CSS v4 + Vanilla CSS
- Zustand (state management)
- TanStack Query v5
- React Router DOM v7
- Lucide React + React Icons
- Framer Motion
- Axios (with CSRF + token refresh interceptors)

**Infrastructure:**
- 3 modules: `/backend`, `/frontend-store`, `/frontend-admin`
